## テキストのデザイン

### テキストを太く（CustomText）

```
<CustomText font="semibold">少し太く</CustomText>
<CustomText font="bold">太く</CustomText>
```

### テキストサイズ（CustomText）

```
<CustomText size="sm">少し小さく</CustomText>
<CustomText size="lg">少し大きく</CustomText>
<CustomText size="xl">大きく</CustomText>
```

### テキストの色（CustomText）

```
<CustomText color="red">赤文字</CustomText>
<CustomText color="green">緑文字</CustomText>
<CustomText color="orange">緑文字</CustomText>
<CustomText color="yellow">緑文字</CustomText>
```

### マーカー（CustomText）

```
// マーカー（CustomText）

<CustomText marker="red">赤マーカー</CustomText>
<CustomText marker="green">緑マーカー</CustomText>
<CustomText marker="orange">オレンジマーカー</CustomText>
<CustomText marker="yellow">黄色マーカー</CustomText>
```

### アンダーマーカー（CustomText）

```
<CustomText underMarker="red">赤アンダーマーカー</CustomText>
<CustomText underMarker="green">緑アンダーマーカー</CustomText>
<CustomText underMarker="orange">オレンジアンダーマーカー</CustomText>
<CustomText underMarker="yellow">黄色アンダーマーカー</CustomText>
```

## デザインコンポーネント

### ボタンコンポーネント（Button）

```

<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>

```
