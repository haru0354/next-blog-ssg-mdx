---
display: true
internalLink: false
images:
 - eyeCatchName: "no_image"
   eyeCatchAlt: "専用画像が用意されてません"
   url: "/"
---

{/* サイドメニュー上部の画像リンク */}

サイドメニューの最上部に表示される画像リンクを表示させるファイルです。

displayを「true」「false」の書き換えによって、サイドメニューの表示が変わります。
true: 表示
false: 非表示

internalLinkを「true」「false」の書き換えによって、内部リンクか外部リンクか指定してください。
true: 内部リンク（next.jsのリンクコンポーネントを仕様）
false: 外部リンク（htmlのaタグを使用 / 新しいタブで開かれる）

記載する内容は下記をご覧ください。

eyeCatchName: 表示する画像ファイル名を拡張子を除いて入力
eyeCatchAlt: 画像のALT
slug: リンク先のURL

記載をしていても「display」が「false」なら表示されないので、メモとして残すこともできます。

{/* 表示する画像を増やす */}

入力する項目をコピー貼り付けで増やして記載してください。

- eyeCatchName: "test1"
  eyeCatchAlt: "test1"
  url: "/test1"
- eyeCatchName: "test2"
  eyeCatchAlt: "test2"
  url: "/test2"
