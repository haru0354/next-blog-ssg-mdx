この記事ではブログアプリを装飾する為の、「デザインのサンプル」を記載しています。

また、どう記述することで、デザインが適用されるかについての記載も行っています。

このブログアプリはMDXの形式で作成されており、「MD」形式での装飾の仕方の他に、デザインが適用されたコンポーネントが作成されており、簡単に記事の装飾が行えるようになっています。

「コピー / 貼り付け」を利用することで、だれでも簡単に装飾された記事を作成が可能です。

## テキストのデザイン

### テキストを太く（CustomText）

props で color を「red」「green」で指定することで変更可能です。

- <CustomText font="semibold">少し太く</CustomText>
- <CustomText font="bold">太く</CustomText>

```
// テキストを太く（CustomText）

<CustomText font="semibold">少し太く</CustomText>
<CustomText font="bold">太く</CustomText>
```

### テキストサイズ（CustomText）

props で size を指定することで変更可能です。

- <CustomText size="sm">少し小さく</CustomText>
- <CustomText size="lg">少し大きく</CustomText>
- <CustomText size="xl">大きく</CustomText>

```
// テキストサイズ（CustomText）

<CustomText size="sm">少し小さく</CustomText>
<CustomText size="lg">少し大きく</CustomText>
<CustomText size="xl">大きく</CustomText>
```

### テキストの色（CustomText）

props で color を「red」「green」で指定することで変更可能です。

- <CustomText color="red">赤文字</CustomText>
- <CustomText color="green">緑文字</CustomText>
- <CustomText color="orange">緑文字</CustomText>
- <CustomText color="yellow">緑文字</CustomText>

```
// テキストの色（CustomText）

<CustomText color="red">赤文字</CustomText>
<CustomText color="green">緑文字</CustomText>
<CustomText color="orange">緑文字</CustomText>
<CustomText color="yellow">緑文字</CustomText>
```

### マーカー（CustomText）

props で marker を「red」「green」で指定することで変更可能です。

- <CustomText marker="red">赤マーカー</CustomText>
- <CustomText marker="green">緑マーカー</CustomText>
- <CustomText marker="orange">オレンジマーカー</CustomText>
- <CustomText marker="yellow">黄色マーカー</CustomText>

```
// マーカー（CustomText）

<CustomText marker="green">緑マーカー</CustomText>
<CustomText marker="orange">オレンジマーカー</CustomText>
<CustomText marker="yellow">黄色マーカー</CustomText>
```

### アンダーマーカー（CustomText）

props で underMarker を「red」「green」で指定することで変更可能です。

- <CustomText underMarker="red">赤アンダーマーカー</CustomText>
- <CustomText underMarker="green">緑アンダーマーカー</CustomText>
- <CustomText underMarker="orange">オレンジアンダーマーカー</CustomText>
- <CustomText underMarker="yellow">黄色アンダーマーカー</CustomText>

## デザインコンポーネント

### ボタンコンポーネント（Button）

props で color を「blue」「red」「green」の指定で変更可能です。

初期値は blue が設定されているので、何も props が無ければ青色になります。

<Button>ボタン</Button>
<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>

```
// ボタンを表示する

<Button>ボタン</Button>
<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>
```

#### ボタンコンポーネント(Button サイズ)

size に props で「small」「normal」「big」で指定することで変更が可能です。

何も指定がなければ「normal」が指定されます。

<Button size="small">ボタン</Button>
<Button size="normal">ボタン</Button>
<Button size="big">ボタン</Button>

```
// ボタンのサイズを変更する

<Button size="small">ボタン</Button>
<Button size="normal">ボタン</Button>
<Button size="big">ボタン</Button>
```

### Box（Box）

children を使用することで利用が可能。
props の「className」に string で tailWind で記述することでスタイルを追加できます。

<Box>box</Box>

スタイルの追加例(rounded)

<Box className="rounded">roundedBox</Box>

スタイルの追加例(border-dashed)

<Box className="border-dashed">border-dashedBox</Box>

```
// Boxデザインを変更する

<Box>box</Box>
<Box className="rounded">roundedBox</Box>
<Box className="border-dashed">border-dashedBox</Box>
```

#### Box（Box タイトル付き）

Box に title で string で props を渡すことで変更をすることができます。

<Box title="タイトル付きBox">テスト</Box>

```
// タイトル付きBox

<Box title="タイトル付きBox">テスト</Box>
```

### FlexBox（FlexBox）

横 2 列の Box。

props の「contents」に配列のオブジェクト形式を渡す。

オブジェクトは「topText」「content」「width」「height」「src」「alt」を追加する。

contentのみ改行ができるようになっています。

但し、「``」でテキストを囲むようにしてください。

<FlexBox
  contents={[
    {
      topText: "TEST",
      content: `
    TEST

    TEST
  `,
      width: 240,
      height: 140,
      src: "/logo.png",
      alt: "a",
    },
    {
      topText: "TEST2",
      content: `
    TEST2

    TEST2
  `,
      width: 240,
      height: 140,
      src: "/logo.png",
      alt: "a",
    },
  ]}
/>

### CustomLink（CustomLink）

mdx 内に設置する「Link」を next.js の「Link コンポーネント」にすることができます。

「children」で「CustomLink」タグを、Next.js の「Link」タグや html の「a」タグの代わりに使用してください。

<CustomLink href="/">TOP PAGE</CustomLink>

```
// CustomLink

<CustomLink href="/">TOP PAGE</CustomLink>
```
