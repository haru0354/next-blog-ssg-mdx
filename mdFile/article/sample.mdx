## テキストのデザイン

### テキストの色（TextColor）

propsでcolorを「red」「green」で指定することで変更可能です。

<TextColor color="red">赤文字</TextColor>
<TextColor color="green">緑文字</TextColor>

```
// テキストの色（TextColor）

<TextColor color="red">赤文字</TextColor>
<TextColor color="green">緑文字</TextColor>
```

### テキストを太く（TextColor）

propsでcolorを「red」「green」で指定することで変更可能です。

<TextColor font="semibold">少し太く</TextColor>
<TextColor font="bold">太く</TextColor>

```
// テキストを太く（TextColor）

<TextColor font="semibold">少し太く</TextColor>
<TextColor font="bold">太く</TextColor>
```


## デザインコンポーネント

### ボタンコンポーネント（Button）

props で color を「blue」「red」「green」の指定で変更可能です。

初期値は blue が設定されているので、何も props が無ければ青色になります。

<Button>ボタン</Button>
<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>

```
// ボタンを表示する

<Button>ボタン</Button>
<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>
```

#### ボタンコンポーネント(Button サイズ)

size に props で「small」「normal」「big」で指定することで変更が可能です。

何も指定がなければ「normal」が指定されます。


<Button size="small">ボタン</Button>
<Button size="normal">ボタン</Button>
<Button size="big">ボタン</Button>

```
// ボタンのサイズを変更する

<Button size="small">ボタン</Button>
<Button size="normal">ボタン</Button>
<Button size="big">ボタン</Button>
```


### Box（Box）

children を使用することで利用が可能。
props の「className」に string で tailWind で記述することでスタイルを追加できます。

<Box>box</Box>

スタイルの追加例(rounded)

<Box className="rounded">roundedBox</Box>

スタイルの追加例(border-dashed)

<Box className="border-dashed">border-dashedBox</Box>

```
// Boxデザインを変更する

<Box>box</Box>
<Box className="rounded">roundedBox</Box>
<Box className="border-dashed">border-dashedBox</Box>
```

#### Box（Box タイトル付き）

Box に title で string で props を渡すことで変更をすることができます。

<Box title="タイトル付きBox">テスト</Box>


```
// タイトル付きBox

<Box title="タイトル付きBox">テスト</Box>
```

### CustomLink（CustomLink）

mdx 内に設置する「Link」を next.js の「Link コンポーネント」にすることができます。

「children」で「CustomLink」タグを、Next.js の「Link」タグや html の「a」タグの代わりに使用してください。

<CustomLink href="/">TOP PAGE</CustomLink>


```
// CustomLink

<CustomLink href="/">TOP PAGE</CustomLink>
```

<BlockquoteInBox contents="a" url="/" linkText="aaa" />

```
// 引用（BlockquoteInBox）
```
