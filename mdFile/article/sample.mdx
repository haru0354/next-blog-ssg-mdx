## デザインコンポーネント

### ボタンコンポーネント（Button）

propsでcolorを「blue」「red」「green」の指定で変更可能です。
初期値はblueが設定されているので、何もpropsが無ければ青色になります。
「children」を利用する。

<Button>ボタン</Button>
<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>

#### ボタンコンポーネント(Button サイズ)

sizeにpropsで「small」「normal」「big」をstringで指定することで変更が可能です。
何も指定がなければ「normal」が指定されます。

<Button size="small">ボタン</Button>
<Button size="normal">ボタン</Button>
<Button size="big">ボタン</Button>

### Box（Box）

childrenを使用することで利用が可能。
propsの「className」にstringでtailWindで記述することでスタイルを追加できます。

<Box>テスト</Box>

スタイルの追加例(rounded)

<Box className="rounded">テスト</Box>

スタイルの追加例(border-dashed)

<Box className="border-dashed">テスト</Box>

#### Box（Box タイトル付き）

Boxにtitleでstringでpropsを渡すことで変更をすることができます。

<Box title="タイトル付きBox">テスト</Box>

### FlexBox（FlexBox）

横2列のBox。
propsの「contents」に配列のオブジェクト形式を渡す
オブジェクトは「topText」「content」「width」「height」「src」「alt」を追加する。

<FlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
]} />

#### ボタンリンク付きFlexBox（FlexBox）

propsで「buttonText」「link」の2つを追加すると、ボタンリンクが追加されます。

<FlexBox contents={[
  { topText: "taaaa", content: "あああああああああああああああああああああああああああああああああああああああああああああああああああああああああああああ", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE", link: "/"},
  { topText: "taaaa2", content: "あああああああああああああああああああああああああああああああああああああああああああああああああああああああああああああ", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE2", link: "/" },
]} />

### 最初の1つ目のみデザインの違う（ThreeFlexBox）

最初の１つ目のみデザインが変わります。
それ以降は横２列になります。
propsの「contents」に配列のオブジェクト形式を渡す
オブジェクトは「topText」「content」「width」「height」「src」「alt」を追加する。

<ThreeFlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
  { topText: "taaaa2", content: "aaaaaaa2", width: 240, height: 140, src: "/logo.png", alt: "a"},
]} />

#### ボタンリンク付き（ThreeFlexBox）

propsで「buttonText」「link」の2つを追加すると、ボタンリンクが追加されます。

<ThreeFlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE", link: "/"},
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE", link: "/"},
  { topText: "taaaa2", content: "aaaaaaa2", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE2", link: "/" },
]} />

### CustomLink（CustomLink）

mdx内に設置する「Link」をnext.jsの「Linkコンポーネント」にすることができます。
「children」で「CustomLink」タグを、Next.jsの「Link」タグやhtmlの「a」タグの代わりに使用してください。

<CustomLink href="/">TOP PAGE</CustomLink>

### 引用（BlockquoteInBox）

propsで「contents」「url」「linkText」を設定する。

<BlockquoteInBox contents="a" url="/" linkText="aaa"  />

### 引用（BlockquoteInBox Titleあり）

propsで「contents」「url」「linkText」を設定する。
propsに「title」を追加

<BlockquoteInBox title="test" contents="a" url="/" linkText="aaa"  />
