## デザインコンポーネント

### ボタンコンポーネント

propsでcolorを「blue」「red」「green」の指定で変更可能です。
初期値はblueが設定されているので、何もpropsが無ければ青色になります。

<Button>ボタン</Button>
<Button color="green">ボタン</Button>
<Button color="red">ボタン</Button>
<Button color="gray">ボタン</Button>

### FlexBox

横2列のBox。
boxの数はmap関数を使用しているので、配列内のオブジェクトの数を増やすことで、数が増えていきます。

<FlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
]} />

#### ボタンリンク付きFlexBox

propsで「buttonText」「link」の2つを追加すると、ボタンリンクが追加されます。

<FlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE", link: "/"},
  { topText: "taaaa2", content: "aaaaaaa2", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE2", link: "/" },
]} />

### 最初の1つ目のみデザインの違うFlexBox

最初の１つ目のみデザインが変わります。
それ以降は横２列になります。
Flex

<ThreeFlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a" },
  { topText: "taaaa2", content: "aaaaaaa2", width: 240, height: 140, src: "/logo.png", alt: "a"},
]} />

#### ボタンリンク付き

propsで「buttonText」「link」の2つを追加すると、ボタンリンクが追加されます。

<ThreeFlexBox contents={[
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE", link: "/"},
  { topText: "taaaa", content: "aaaaaaa", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE", link: "/"},
  { topText: "taaaa2", content: "aaaaaaa2", width: 240, height: 140, src: "/logo.png", alt: "a", buttonText: "TOP PAGE2", link: "/" },

### CustomLink

mdx内に設置する「Link」をnext.jsの「Linkコンポーネント」にすることができます。
「CustomLink」タグをNext.jsの「Link」タグやhtmlの「a」タグの代わりに使用してください。

<CustomLink href="/">TOP PAGE</CustomLink>
