# 補足

name が記載されていないと、グローバルメニューは表示されないようにしています。
そのため、グローバルメニューを表示するのであれば、まずは 1 つ目の name と slug を記載してください。

## slug

slug は最初の「/」はいりません。
「/sample」なら「sample」を明記してください。

# name

10 文字を目安に設定してください。

幅 375px のデバイスで 12 文字まで 1 行で表示されます。
それ以降は 2 行での表示となります。

各項目を 2 行にできるように slice で文字数の制限はしていません。
